## HTML & CSS Practice (Node.js + VSCode)

VSCode 환경과 Node.js 개발 환경을 기반으로 진행한 **HTML, CSS 실습 프로젝트**입니다.  
웹 페이지 구조와 스타일링 기초를 익히고, 정적인 웹 페이지를 직접 만들어보는 연습을 합니다.

## 🚀 기술 스택
- Node.js (실습 환경 세팅용)
- VSCode
- HTML5
- CSS3

## 📝 주요 학습 내용
- HTML 문서 구조와 시맨틱 태그
- 텍스트, 링크, 이미지 삽입
- CSS 선택자와 스타일링 기초
- 레이아웃 구성 (Flexbox, Grid 기초)
- 간단한 정적 웹 페이지 제작
